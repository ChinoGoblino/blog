Some of these are widely known; some aren’t. All of them are in my PATH on every machine I use.

fd

A replacement for find that’s faster and has saner defaults. The syntax is fd <pattern> rather than find . -name <pattern>, which saves a surprising amount of typing over a day.

# find all markdown files in the current tree
fd '\.md$'

# find and delete build artifacts
fd -e o -x rm {}

ripgrep

grep but faster and with better defaults: recursive by default, respects .gitignore, colors the output, and shows context without extra flags.

# search for a function name across a project
rg 'fn handle_request'

# search only in certain file types
rg --type py 'import asyncio'

rg is fast enough that I rarely think about it. It’s one of those tools that disappears into the background.

jq

JSON on the command line. Indispensable when working with APIs or any log format that emits JSON.

# pretty-print
curl -s https://api.example.com/data | jq .

# extract a field from each element of an array
cat events.json | jq '.[].timestamp'

# filter and transform
cat users.json | jq '[.[] | select(.active == true) | {id, email}]'

The syntax takes an hour to learn and then becomes muscle memory.

zoxide

A smarter cd. It learns which directories you visit frequently and lets you jump to them with abbreviated names.

z proj        # jumps to ~/code/projects or wherever you spend time
z conf nix    # narrows to the most frequent directory matching both terms

After a week it’s faster than tab-completing long paths.

bat

cat with syntax highlighting and line numbers. Useful when quickly reviewing a file without opening an editor. Also has a --plain flag for when you need clean output.

bat config.yaml
bat --plain script.sh | pbcopy

A note on tool accumulation

There’s a failure mode where you spend more time installing and configuring tools than actually using them. The test I use: did I reach for this tool three times this week without thinking about it? If yes, it earns a place. If I have to remind myself it exists, it probably isn’t solving a real problem for me.

These five pass that test. Your list will be different.
